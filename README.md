# 🧪 ANASOL Web Application

A modern and responsive corporate website for **Laboratoire ANASOL**, a Tunisian geotechnical laboratory specializing in soil analysis, geotechnical studies and quality control for construction and public works.

---

## 📖 Overview

The ANASOL Web Application is a Single Page Application (SPA) developed with **Angular 20**.

Its purpose is to present the laboratory, showcase its expertise and projects, and provide an easy way for clients to get in touch.

The application has been designed with a clean, modern interface while remaining fully responsive across desktop, tablet and mobile devices.

---

## ✨ Features

* Responsive modern design
* Sticky navigation bar
* Mobile navigation drawer
* Smooth scrolling between sections
* Company presentation
* Services / Activities section
* Projects showcase
* Client testimonials carousel
* Contact form with Angular Material validation
* Embedded Google Maps location
* Scroll-to-top floating button
* Default redirect page for unavailable features
* Automatic deployment to GitHub Pages using GitHub Actions

## 🚀 Development

Clone the repository

```bash
git clone https://github.com/akramBNA/anasol-web-app.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
ng serve
```

Navigate to

```
http://localhost:4200
```

---

## 🏗️ Production Build

```bash
ng build
```

The production files will be generated inside the `dist/` directory.

---

## 🌐 Deployment

This project is automatically deployed to **GitHub Pages** using **GitHub Actions** whenever changes are merged into the `main` branch.
